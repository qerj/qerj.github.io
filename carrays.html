Think of arrays as simple unions. They are close together in memory.

here we make a array of type int arrayt with the lenght of 5 * 4 bytes, 5 elements and 4 bytes each element.

we then make a variable equal to this array as a integer cast..

allowing us to do arithmetic on the array poo = poo + 1 * 4

this is sort of what this looks like when a program is compiled and ran

we first got the base address which is poo which is the address of arrayt. And then we add one which is our index so it would be

arrayt[1]. We then have 4 which is our byte scaler value. Since this is a interger it's size is 4 bytes long.

we then cast back poo to a pointer and dereference the pointer to get the value at arrayt[1].

with this done the value of the address at arrayt[1] is edited! Printing this out gives out 5!
